# python-eda-outlier-detection
Exploratory Data Analysis (EDA) in Python using Pandas, NumPy, and Matplotlib. Includes data summary, missing value analysis, distribution plots, IQR-based outlier detection and handling, correlation analysis, and cleaned dataset export.
## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab

## Key Steps
- Dataset loading and inspection
- Descriptive statistics
- Missing value analysis
- Histogram and boxplot visualization
- IQR-based outlier detection
- Outlier handling using capping
- Correlation analysis
- Export of cleaned dataset
